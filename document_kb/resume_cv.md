# Resume / CV

## Document Details
* **Document Name**: Resume
* **Alternative Names**: Curriculum Vitae, CV, Professional Resume, Bio-data
* **Primary Uses**: Educational History, Professional Background, Employment details, Skills verification
* **Issuing Authority**: Self-generated / Professional document

## Fields and Information Contained
* **Work Experience**: Job titles, company names, employment durations, job descriptions, responsibilities.
* **Skills**: Technical skills (e.g., "Python", "Java", "React"), soft skills, certifications, tools, languages.
* **Education History**: Degrees (e.g., "Bachelor of Technology", "Master of Science"), school/university names, years of graduation, GPA, CGPA, grades.
* **Projects**: Project names, descriptions, tech stacks, GitHub links, project outcomes.
* **Contact Information**: Phone number, professional email address, LinkedIn profile URL, GitHub profile URL, portfolio website.
* **Professional Summary**: Short bio or objective.
