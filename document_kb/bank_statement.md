# Bank Statement / Passbook

## Document Details
* **Document Name**: Bank Statement
* **Alternative Names**: Bank Passbook, Account Statement, Transaction Summary, Bank Certificate
* **Primary Uses**: Proof of Address (PoA), Proof of Income, Financial Viability / Assets Verification
* **Issuing Authority**: Licensed Bank / Financial Institution

## Fields and Information Contained
* **Account Holder Name**: Full name of the primary/joint account holder.
* **Account Number**: Unique bank account number.
* **IFSC Code / Swift Code / Branch Details**: Bank identifier codes.
* **Registered Address**: Address of the holder registered with the bank (used as Proof of Address).
* **Transactions / Balance**: Record of deposits, withdrawals, and current closing balance.
