# Voter ID Card

## Document Details
* **Document Name**: Voter ID Card
* **Alternative Names**: Electoral Photo Identity Card (EPIC), Election Card, Voter Card
* **Primary Uses**: Proof of Identity (PoI), Proof of Address (PoA), Proof of Age
* **Issuing Authority**: Election Commission of India (ECI)

## Fields and Information Contained
* **EPIC Number / Voter ID Number**: Alphanumeric unique identifier. (e.g., "EPIC No", "Voter ID Number", "Elector's Photo Identity Card Number").
* **Elector's Name / Full Name**: Legal name of the voter.
* **Father's Name / Mother's Name / Husband's Name**: Relation name of the voter.
* **Gender**: Male, Female, or Other.
* **Date of Birth / Age**: Either the exact date of birth or the age of the elector on a specific qualifying date.
* **Full Address**: Residential address of the elector, including House Number, Village/Mohalla, Town/City, Post Office, District, State, and Pin Code.
* **Assembly Constituency**: Electoral constituency name and number.
